🔧 Coze API 错误排查工具

根据官方文档快速诊断和解决API调用问题

📋 常见错误码说明

HTTP 401 - 未授权

401 Unauthorized
API密钥无效、已过期或格式错误

解决方案：

  • 检查Personal Access Token是否正确
  • 确认令牌格式为：Bearer pat_xxxxxxxxxx
  • 验证令牌是否已过期
  • 确保令牌具有chat权限

HTTP 403 - 禁止访问

403 Forbidden
权限不足，令牌缺少必要权限

解决方案：

  • 确保PAT令牌具有chat权限
  • 检查机器人是否属于令牌对应的空间
  • 验证机器人是否已发布为API服务

HTTP 404 - 资源不存在

404 Not Found
机器人ID不存在或未发布

解决方案：

  • 检查机器人ID是否正确
  • 确认机器人已发布为API服务
  • 验证机器人与令牌在同一空间

HTTP 429 - 请求过多

429 Too Many Requests
请求频率超过限制

解决方案：

  • 降低请求频率
  • 实现请求重试机制
  • 考虑升级API配额

Coze 错误码 702242001

Parameter passing error
参数传递错误，请参考对应API文档

解决方案：

  • 检查请求头是否完整（Content-Type, Authorization等）
  • 验证请求体格式是否正确
  • 确认所有必需参数都已提供
  • 检查参数类型和格式
