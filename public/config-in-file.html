配置说明

现在API配置直接保存在HTML文件中的JavaScript变量里，不再使用浏览器缓存。 这意味着配置会永久保存在文件中，即使清除浏览器缓存也不会丢失。

如何设置配置

方法1： 在主页面点击设置按钮，在弹出的配置面板中填写API信息

方法2： 直接编辑 index(5).html 文件，在 API_CONFIG 变量中填写：

let API_CONFIG = {
    apiKey: '您的API密钥', // 在这里填写
    botId: '您的机器人ID'   // 在这里填写
};
